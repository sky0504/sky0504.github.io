安装中文包

文件->首选项->扩展（快捷键快速定位Ctrl+Shift+X），然后输入Chinese Install.

文件中文乱码处理

文件->首选项->设置，然后搜素files.autoGuessEncoding为True

必备插件

  • vscode-icons 文件图标,根据文件图标快速的了解文件类型
  • Beautify 代码格式化工具
  • Code Runner 万能运行工具
  • Code Spell Checker 拼写检查程序

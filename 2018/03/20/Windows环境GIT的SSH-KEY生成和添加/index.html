Windows环境GIT的SSH-KEY生成和添加

产生KEY的相关文件位置变更

如果不需要更改key默认生成位置和相关文件请跳过本步骤

文字描述

  1. 在桌面我的电脑右击，选择最下边的属性，同样也可以打开系统属性界面(快捷键windwos+Break组合键)
  2. 点击高级系统设置，接着点击环境变量
  3. 打开环境变量编辑界面后，点击新建，添加一个新的系统变量HOME，值为自己想设定的路径。最后点击确定，确定，确定。完成环境变量的设置。

图片展示

生成和添加KEY

文字描述

  1. 输入命令ssh-keygen -t rsa -C "bluesky0504@163.com"，然后点击回车
  2. 根据提示的文字，输入你的结果。一般取默认路径和不设置密码，所以连续按三次回车，即可出现生成成功的结果
  3. 成功之后，输入命令clip < id_rsa.pub然后回车来拷贝产生的key
  4. 打开你要添加的key的位置，然后添加上去
  5. 最后添加完成之后，在本机上输入ssh -T git@xxxx.com，查看是否真正的添加成功。

    第一项的`bluesky0504@163.com换成自己的邮件地址;第三步中的id_rsa.pub是生成的id_rsa.pub文件所在路径;最后一步的xxxx.com`要换成自己的服务器地址
